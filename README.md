# Android Tooltips System

## 1. Введение
Система подсказок (Tooltips) в Android приложениях предназначена для улучшения пользовательского опыта. Подсказки предоставляют контекстную информацию о элементах интерфейса, позволяя пользователю быстро понять функциональность без обращения к документации.

### 1.1 Основные подходы к реализации
- **Material Design Tooltip**  
  Использование стандартных компонентов AndroidX для подсказок в стиле Material Design.
- **Кастомные реализации**  
  Создание собственных тултипов с помощью `PopupWindow`, `DialogFragment` или кастомных `View`.
- **Атрибуты доступности**  
  Использование стандартных атрибутов `accessibility` для базовых подсказок.

### 1.2 Ключевые принципы
- **Контекстность:** Подсказка должна соответствовать элементу интерфейса.
- **Краткость:** Текст лаконичный и понятный.
- **Время показа:** 2–5 секунд оптимально.
- **Позиционирование:** Логичное расположение относительно элемента.
- **Доступность:** Поддержка скринридеров и средств доступности.

## 2. Техническое задание

### 2.1 Цель проекта
Создание универсальной системы подсказок для Android приложения с использованием Kotlin и Android Studio.

### 2.2 Функциональные требования

#### 2.2.1 Базовый функционал
- Отображение текстовых подсказок для UI элементов.
- Поддержка различных позиций отображения: сверху, снизу, слева, справа.
- Автоматическое скрытие через заданное время.
- Возможность ручного закрытия.
- Анимация появления/исчезновения.

#### 2.2.2 Расширенный функционал
- Поддержка HTML-форматирования в тексте.
- Кастомизация внешнего вида: цвета, шрифты, скругления.
- Цепочки подсказок (tour guide).
- Сохранение состояния показанных подсказок.
- Поддержка жестов для навигации.

### 2.3 Нефункциональные требования
- Минимальная версия Android: API 21 (Android 5.0)
- Язык программирования: Kotlin
- Архитектура: MVVM
- Используемые библиотеки: AndroidX, Material Design Components

### 2.4 Технические спецификации
**Компоненты системы:**
- `TooltipManager` – управление жизненным циклом подсказок.
- `TooltipView` – кастомная View для отображения.
- `TooltipPreferences` – хранение состояния показа.
- `TooltipAnimation` – управление анимациями.
